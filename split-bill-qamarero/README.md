# Split Bill Qamarero

## Requisitos

- Node.js v18+
- npm o yarn

## Instalación

```bash
npm install
npm install ws
```

## Arrancar el servidor de WebSockets

node src/server/server.js

Esto levanta el servidor que sincroniza las selecciones de productos en tiempo real

## Arrancar la app React

npm run dev

La app se abrirá en el navegador y se conectará al servidor de WebSockets
